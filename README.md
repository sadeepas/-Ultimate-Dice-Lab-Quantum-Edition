## ‚ú® Features

*   **Customizable Dice Rolls**: Choose from a variety of dice types (d4, d6, d8, d10, d12, d20) and roll up to five dice at once.

## üöÄ How to Use

1.  Open the `dice.html` file in your web browser.
2.  Use the **Controls** panel to select your desired dice type, number of dice, and theme.
3.  Click the **Roll** button or press the 'R' key to roll the dice.
4.  To run a large number of trials, enter the desired number in the **Simulation & Data** panel and click **Run Simulation**.
5.  Explore the **Statistics**, **Frequency Tally**, and **Frequency Distribution** panels to analyze the results.
6.  In **Casino Mode**, place bets and see if you can increase your balance.
